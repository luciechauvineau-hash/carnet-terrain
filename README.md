# Carnet de Terrain 🌿

Outil d'éducation à l'environnement pour animateurs de séjours ados.
Fonctionne **hors ligne**, s'installe sur le téléphone comme une appli.

---

## Mettre en ligne en 5 étapes (GitHub Pages)

### Étape 1 — Créer un compte GitHub
Va sur https://github.com et crée un compte gratuit.

### Étape 2 — Créer un nouveau dépôt
- Clique sur le bouton vert "New" (ou "Nouveau dépôt")
- Nom du dépôt : `carnet-terrain` (ou ce que tu veux)
- Coche "Public"
- Clique "Create repository"

### Étape 3 — Uploader les fichiers
- Dans ton nouveau dépôt, clique "uploading an existing file"
- Glisse-dépose tous les fichiers de ce dossier :
  - index.html
  - manifest.json
  - sw.js
  - icon-192.svg
  - icon-512.svg
- Clique "Commit changes"

### Étape 4 — Activer GitHub Pages
- Va dans "Settings" (Paramètres) du dépôt
- Dans le menu gauche, clique "Pages"
- Sous "Source", sélectionne "Deploy from a branch"
- Branche : "main", dossier : "/ (root)"
- Clique "Save"

### Étape 5 — Récupérer l'URL
- Après 1-2 minutes, l'URL apparaît en haut de la page Pages
- Elle ressemble à : `https://ton-pseudo.github.io/carnet-terrain`
- **Envoie cette URL aux animateurs par WhatsApp ou mail**

---

## Installation sur téléphone

**iPhone / iPad :**
1. Ouvre l'URL dans Safari
2. Appuie sur le bouton Partager (carré avec flèche)
3. Sélectionne "Sur l'écran d'accueil"
4. Appuie "Ajouter"

**Android (Chrome) :**
1. Ouvre l'URL dans Chrome
2. Appuie sur les 3 points en haut à droite
3. Sélectionne "Ajouter à l'écran d'accueil"
4. Appuie "Ajouter"

L'appli est maintenant installée et fonctionne **sans connexion internet**.

---

## Données & sécurité

- Les données sont sauvegardées **localement sur chaque téléphone**
- Rien n'est envoyé sur internet
- L'animateur exporte ses fiches depuis l'onglet Journal pour les partager
- Poids total de l'appli : moins de 100 Ko
